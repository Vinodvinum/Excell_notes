Learning Path

🧱 10 Phases to Professional Analyst

Structured progression from formulas → dashboards → automation → business analysis.

🧱 Phase 1: Advanced Formulas

Master the tools analysts use daily:

  • Logical functions (IF, IFS, IFERROR)
  • XLOOKUP (most important lookup)
  • INDEX + MATCH (advanced matching)
  • Dynamic arrays (FILTER, UNIQUE, SORT)

Real Task: Match employee salaries across 3 sheets with 50k rows.

🔧 Phase 2: Data Cleaning

Handle messy real-world data:

  • Remove duplicates & null values
  • Standardize formats
  • Extract text patterns
  • Advanced conditional formatting

Real Task: Clean CRM data with inconsistent formats before analysis.

📊 Phase 3: Pivot Analytics

Core analyst reporting tool:

  • Multi-level pivots
  • Calculated fields & items
  • Show values as percentages
  • Running totals & trends

Real Task: Analyze revenue by department, region, product category.

📈 Phase 4: Dashboard Engineering

Build executive-level dashboards:

  • KPI design & tracking
  • Professional visualizations
  • Dynamic filtering (slicers)
  • Interactive charts

Real Task: Build sales dashboard for management with KPIs and trends.

⚙️ Phase 5: Power Query ETL

Automation for data professionals:

  • Import multiple files
  • Clean & merge automatically
  • Schedule monthly refresh
  • Reduce manual work 80%

Real Task: Auto-consolidate 50 monthly sales files into one report.

🧩 Phase 6: Power Pivot & DAX

Advanced analytics:

  • Data modeling
  • Relationships & dimensions
  • DAX calculations
  • Time intelligence

Real Task: Build "Year-over-Year" comparison KPI with DAX.

🤖 Phase 7: Excel Automation

Save hours with macros & VBA:

  • Record & edit macros
  • VBA fundamentals
  • Automate report formatting
  • One-click MIS system

Real Task: Build 1-click dashboard refresh that runs monthly reports.

🧠 Phase 8: Business Analysis

Think like an analyst:

  • Variance analysis (Budget vs Actual)
  • Trend identification
  • Forecasting & scenarios
  • Performance insights

Real Task: Explain why sales dropped 20% and what to do.

🚀 Phase 9: Real Projects

Portfolio-building projects:

  • Sales Dashboard
  • HR Analytics System
  • Financial Analysis
  • Automated MIS Engine

Real Task: Build complete dashboard + automation system.

💼 Phase 10: Interview Prep

Get hired as analyst:

  • Common interview questions
  • Excel tests & case studies
  • Portfolio presentation
  • Salary negotiation

Real Task: Ace technical interview with real analyst scenarios.
