# Excel Notes — Data Analyst Learning

Comprehensive Excel learning material for data analysts, covering 18 chapters across 6 phases.

## 📊 Phases

- **Phase 1**: Foundation (Chapters 1-3) - Basics & Interface
- **Phase 2**: Data Handling (Chapters 4-6) - Sorting, Filtering, Formulas
- **Phase 3**: Functions (Chapters 7-9) - Logical, Lookup, Text Functions
- **Phase 4**: Analysis Tools (Chapters 10-12) - Pivot Tables, Charts, Formatting
- **Phase 5**: Advanced Skills (Chapters 13-15) - Data Validation, What-If, Shortcuts
- **Phase 6**: Real Projects (Chapters 16-18) - Dashboard, Storytelling, Interview Prep

## 🎯 How to Use

1. Visit the main page: `index.html`
2. Select a chapter from the tabs
3. Each chapter includes:
   - Business situation explanation
   - Step-by-step breakdown
   - Real examples
   - Interview Q&A
   - Practice questions with answers
