Example Teaching Style

Business problem: HR has employee IDs in Sheet A and salaries in Sheet B. How do we merge safely for 20,000 rows?

Then we explain why copy-paste fails, why lookup functions are used, which method is faster, and how to make dynamic reporting logic.

10-Phase Roadmap (Simple Story View)

Phase 1: Advanced Formulas

We start with logical functions, lookup matching, and dynamic arrays.

IFIFSXLOOKUPINDEX/MATCHFILTERUNIQUE

Scenario: Categorize KPI status and link employee, product, and customer data across multiple tables.

Phase 2: Data Cleaning and Transformation

Learn clean reporting foundations with real messy datasets.

TRIMCLEANSUBSTITUTETEXTSPLIT

Scenario: Fix inconsistent CRM export before dashboard build.

Phase 3: Pivot Table Analytics

Master multi-level analysis, percentage views, and trend summaries.

Scenario: Revenue trend analysis by region and department for monthly review meeting.

Phase 4: Dashboard Engineering

Design management dashboards with storytelling and KPI flow.

Scenario: Build an executive sales dashboard with clear action points.

Phase 5: Power Query (Must Master)

Automate ETL workflows: merge, append, reshape, unpivot.

Scenario: Monthly MIS automation from multiple input files.

Phase 6: Power Pivot and Data Modeling

Build relationships, star schema basics, and DAX measures.

CALCULATESUMXFILTERTOTALYTD

Phase 7: Excel Automation

Use Macros and VBA to reduce manual report work.

Mini project: One-click MIS reporting and dashboard refresh.

Phase 8: Business Analysis in Excel

Focus on decision making: variance, forecasting, bottlenecks.

Scenario: Explain sales drop with root cause and recommendation.

Phase 9: Advanced Analyst Projects

  • Executive Sales Dashboard
  • Automated MIS Reporting System
  • HR Analytics Dashboard
  • Financial Analysis Dashboard

Phase 10: Interview Preparation

Practice technical questions, case studies, and dashboard explanation.

Focus: XLOOKUP vs INDEX MATCH, Power Query workflows, KPI logic, storytelling.
