Phase 1: Foundation — Basics

Chapter 1: What is Excel & Why It's Used

🧠 What is Excel?

Excel is a spreadsheet tool used for data storage, cleaning, analysis and visualization.

Excel = Fast + Flexible + Easy

🏢 Real Story (Amazon Analyst)

You work at Amazon. Manager says: "Quickly analyze this data"

order_id   amount    city
1          5000      Bangalore
2          3000      Mysore

You don't open Power BI. You don't write SQL. You open Excel because it's quick.

✅ Why Excel is Used

  • Quick Analysis: No setup required
  • Data Cleaning: Remove duplicates, fix data
  • Calculations: SUM, AVERAGE formulas
  • Dashboards: Charts and reports

📊 Excel vs SQL vs Power BI

Tool Use Case
Excel Quick work, small/medium data
SQL Large databases, complex queries
Power BI Visual dashboards, presentations

⚡ When to Use Excel

  • Data is small to medium
  • Need quick insights
  • Need manual data work
  • Working alone (not team dashboard)
Answer: Excel is used for quick analysis of small datasets, data cleaning, and fast insights without setup.

🎤 Interview Answer

"What is Excel?"

Excel is a spreadsheet tool used for data storage, cleaning, analysis and visualization. It is commonly used for quick data analysis and reporting, especially with small to medium datasets.

🧪 Practice Questions

Q1: Why use Excel instead of Power BI?
Ans: Excel is faster for quick analysis. Power BI is better for large datasets and team dashboards.
Q2: What can Excel do?
Ans: Store data, clean it, calculate metrics, create charts and dashboards.

Chapter 6: Basic Formulas (Core)

🧮 Basic Functions

SUM - Add all values

=SUM(B2:B10)
Result: 50000

AVERAGE - Mean value

=AVERAGE(B2:B10)
Result: 5000

COUNT - Count numbers

=COUNT(B2:B10)
Result: 10

COUNTA - Count non-empty cells

=COUNTA(A2:A10)

MIN - Smallest value

=MIN(B2:B10)
Result: 1000

MAX - Largest value

=MAX(B2:B10)
Result: 10000

📊 Example Dashboard

Answer: Basic formulas: SUM for totals, AVERAGE for mean, COUNT for count, MIN/MAX for extremes.

Phase 3: Functions — Real Analysis

Chapter 7: Logical Functions

🔀 IF Statement

Syntax

=IF(condition, if_true, if_false)

Example

=IF(B2>5000, "High", "Low")
If amount > 5000: "High", else "Low"

AND Function

=IF(AND(B2>5000, C2="Bangalore"), "Yes", "No")
Both conditions must be true

OR Function

=IF(OR(B2>5000, C2="Mumbai"), "Yes", "No")
Any condition can be true

NOT Function

=IF(NOT(C2="Delhi"), "Yes", "No")
NOT equals
Answer: IF for conditions, AND for all true, OR for any true, NOT for negation.

Chapter 8: Lookup Functions (VERY IMPORTANT)

🔍 XLOOKUP (Modern)

Syntax

=XLOOKUP(lookup_value, lookup_array, return_array)

Example

Product data in another sheet:
product_id | product_name
1          | Laptop
2          | Mouse

Formula: =XLOOKUP(1, product_id, product_name)
Result: Laptop

VLOOKUP (Classic)

=VLOOKUP(lookup_value, table_array, col_index, FALSE)

Example

=VLOOKUP(1, products_table, 2, FALSE)
Look for 1 in first column, return column 2

HLOOKUP (Horizontal)

=HLOOKUP(value, table, row_index, FALSE)
Same as VLOOKUP but searches horizontally
Answer: XLOOKUP best (modern), VLOOKUP for vertical lookup, HLOOKUP for horizontal.

🧪 Practice

Q1: How to lookup product name by ID?
Ans: =XLOOKUP(ID, product_IDs, product_names)

Chapter 9: Text Functions

📝 Text Manipulation

LEFT - First N characters

=LEFT("Bangalore", 3)
Result: "Ban"

RIGHT - Last N characters

=RIGHT("Bangalore", 4)
Result: "lore"

MID - Middle characters

=MID("Bangalore", 2, 4)
Result: "anga"

CONCAT - Combine text

=CONCAT("Hello", " ", "World")
Result: "Hello World"

CONCATENATE - Old version

=CONCATENATE(A1, " ", B1)

Example: Full Name

First Name: John
Last Name: Doe
=CONCAT(A2, " ", B2)
Result: "John Doe"
Answer: LEFT/RIGHT/MID extract text, CONCAT joins strings.

Phase 4: Analysis Tools

Chapter 10: Pivot Tables (MOST IMPORTANT)

🔄 What is Pivot Table?

Tool to summarize data without formulas. Drag, drop, and aggregate.

When to Use

  • Quick grouping (by month, city, category)
  • No formula knowledge needed
  • Fast aggregations (SUM, COUNT, AVG)

📊 Step-by-Step

  1. Select Data: Click anywhere in table
  2. Insert → PivotTable: Click on menu
  3. Choose Location: New sheet or existing
  4. Drag Fields:
    • Row Labels: City, Product
    • Columns: Month, Year
    • Values: Amount, Count (auto-summed)
    • Filters: Optional filters
  5. Analyze: Read summary

Example: Sales by City

Original data: 1000 rows
Pivot result:
City       | Total Sales
Bangalore  | ₹150,000
Mumbai     | ₹200,000
Delhi      | ₹100,000

Pivot Tricks

  • Right-click value → Change aggregation (SUM, AVG, COUNT)
  • Add multiple fields to rows/columns
  • Refresh data: right-click → Refresh
Answer: Pivot tables: drag fields, auto-aggregate. Fastest way to summarize data in Excel.

🧪 Practice

Q1: How to create pivot table?
Ans: Select data → Insert → PivotTable → drag fields

Chapter 17: Storytelling in Excel

🎤 How to Present Insights

Story Structure

  1. Headline: Main finding (e.g., "Sales up 20% this quarter")
  2. Context: Why it matters
  3. Evidence: Charts, numbers
  4. Action: What to do next

Example Narrative

"Our Q1 sales reached ₹500K, up 20% from Q4.
Growth came from Bangalore region (+40%).
Mumbai sales declined (-10%).

Recommendation: Expand Bangalore operations,
investigate Mumbai decline."

Dashboard Tips

  • Title: Clear statement
  • Summary box: Key findings in words
  • Charts: Visual proof
  • Recommendation: What to do
Answer: Story: headline + context + evidence + action. Make it business-focused, not technical.
